## 页面
1. 首页
2. 分类
3. 留言
4. 友链
5. 记录
6. 关于
7. 项目
8. 登录

## 如何使用？

前端
``` 
git clone https://github.com/HryYyR/blog.git 
 ```
 
```
npm i 
 ```
 
```
npm run dev
 ```
 
后端
```
git clone https://github.com/HryYyR/blognode.git 
 ```
 
```
npm i 
 ```
 
```
node index.js 
or
nodemon index.js
 ```
